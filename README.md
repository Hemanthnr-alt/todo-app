🚀 30 - Task Manager App

A modern full-stack productivity app built with React, Node.js, and PostgreSQL, converted into a mobile app using Capacitor.

✨ Features
✅ Task management (Create, Edit, Delete)
📅 Calendar view
🧠 AI Assistant for smart productivity
📂 Categories & organization
🔐 Authentication (JWT-based)
📱 Android APK support (via Capacitor)
🌐 Deployed backend & frontend
⚡ Offline detection support
🛠️ Tech Stack
Frontend
React (Vite)
CSS (Custom UI + Glassmorphism)
Axios
Backend
Node.js
Express.js
PostgreSQL
JWT Authentication
Mobile
Capacitor (Android)
Deployment
Frontend → Vercel
Backend → Render

⚙️ Setup Instructions
1. Clone the repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
2. Install dependencies
cd frontend
npm install

cd ../backend
npm install
3. Setup environment variables

Create .env in backend:

DATABASE_URL=your_db_url
JWT_SECRET=your_secret
4. Run locally

Frontend:

npm run dev

Backend:

npm start
📦 Build APK
npm run build
npx cap sync android

Then open in Android Studio:

Build → Build APK(s)
